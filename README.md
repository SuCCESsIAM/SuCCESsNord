# SuCCESs-Nord

SuCCESs-Nord is a Nordic version of the global [SuCCESs IAM](https://github.com/SuCCESsIAM/SuCCESsIAM), which is described in [this paper](https://gmd.copernicus.org/articles/18/4805/2025/).

SuCCESs-Nord has the same structure as the global model, but it represents separately Denmark, Norway, Sweden, Finland, and the Baltics. A global region (i.e. the global model) is provided as a periphery region for the Nordic regions that represents the e.g. global energy markets and GHG emissions.
